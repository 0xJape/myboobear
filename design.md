💌 DESIGN.md

Project Title: Love Letter Website
Concept: Interactive romantic message experience with music

🎯 Overview

This project is a single-page interactive website that delivers a heartfelt love message through a cinematic, suspense-driven experience.

The user journey mimics opening a private love letter, enhanced with soft background music to deepen emotional impact.

💡 Core Experience Flow
1. Landing State (Suspense Intro)
Dark, elegant background
Centered animated envelope
Soft glow / pulse animation
Music is OFF initially (browser restriction)

👉 Small hint text:

“Tap the envelope…”

2. First Interaction (IMPORTANT)
User clicks the envelope
Actions triggered:
Envelope opens animation
Music starts playing 🎵
Transition begins (fade + zoom)
3. Transition Scene
Smooth cinematic effects:
Fade to soft background
Slight zoom-in
Blur → clear
4. Message Reveal
Message appears using:
Typing animation OR
Line-by-line fade-in
Final line:
Bigger font
Glowing or highlighted 💖
5. Ending State
Buttons:
“Replay 💌” → restart experience
“Pause Music 🎵” → toggle audio
